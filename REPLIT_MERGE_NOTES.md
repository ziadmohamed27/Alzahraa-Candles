# SIRAJ v13 — Replit-inspired frontend merge

This version keeps the existing static Netlify/Supabase website architecture and safely ports the useful visual ideas from the Replit React prototype.

## Added
- Replit-inspired editorial luxury hero composition.
- Real Supabase WebP candle image in the hero.
- Premium product-card styling: improved shadows, image ratio, badge treatment, hover polish.
- New luxury gift-experience section.
- New testimonials section.
- Stronger ivory/espresso/bronze visual system.
- Better section rhythm, dividers, card depth, and footer mood.

## Preserved
- Existing Supabase connection.
- Existing product loading from the `products` table.
- Cart, WhatsApp checkout, wishlist logic, auth/account/orders/admin pages.
- Existing file structure for Netlify deployment.

## Note
This is not a full conversion to the Replit React app. It intentionally avoids replacing the working project architecture to reduce risk.
